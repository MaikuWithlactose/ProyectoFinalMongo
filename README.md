## Angular Profesionales
  ### How to run
  - cd profesionalesApp
  - npm i (si hay algún problema se podría utilizar: npm install --legacy-peer-deps ya que evita los problemas de de restricciones de las últimas versiones de npm según he estado leyendo)
  - ng serve
